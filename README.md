# Portfolio Project

This portfolio showcases selected digital projects and ongoing learning initiatives by Maria Melo, based in Portugal.

The website highlights hands-on front-end projects built with modern web technologies, reflecting a growing interest in digital experiences, user interaction, and problem-solving through technology.

## Project Structure

The project consists of the following files and directories:

- **index.html**: The main entry point of the portfolio website, containing the structure and content for the homepage, including navigation, header, and footer.
- **about.html**: Provides information about Maria Melo, detailing her background and skills.
- **projects.html**: Showcases the projects completed by Maria Melo, highlighting her work as a frontend developer and UX designer.
- **contact.html**: Includes a contact form or information for users to reach out to Maria Melo.
- **styles/style.css**: Contains the main styles for the portfolio website, including layout, typography, and color schemes.
- **styles/collage.css**: Dedicated to styling the collage items and images, ensuring they are responsive and maintain earthy, brown tones.
- **scripts/main.js**: Contains JavaScript code for interactivity and functionality on the website, such as handling navigation or form submissions.
- **data/pinterest.json**: Contains structured data related to the Pinterest boards and pins saved, which can be used to dynamically generate content on the website.
- **images**: Directory holding image files used throughout the portfolio, including hero images and project visuals.

## Features

- Responsive design that adapts to various screen sizes.
- Earthy, brown tones used throughout the site for a cohesive aesthetic.
- Interactive elements powered by JavaScript for enhanced user experience.
- Clear navigation to easily access different sections of the portfolio.
